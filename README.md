# Guess country

### The classic guess the country game with some extra features.

In this project, I built a game that renders some random flags of countries from API (https://restcountries.eu) and the user should guess the country by 6 attempts.

An App built with:

- HTML
- SASS
- JS (OOP, ES6, modules...)

To run this project use npm:
npm install
npm run start

Production build:
npm run build
